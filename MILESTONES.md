# Peliguard Website Cleanup & Fix Milestones

## Current Problems Identified
- **Text is "barfed up"** - No visual organization or containers
- **No visual hierarchy** - All sections blend together on white background
- **No section separation** - Hero, competencies, and newsletter all look the same
- **Poor content organization** - Text is just stacked without visual structure
- **Missing visual containers** - Cards float on white space without context
- **No clear flow** - Sections don't guide the eye naturally

---

## Milestone 1: Visual Section Separation & Backgrounds
**Goal:** Create clear visual distinction between sections using backgrounds and containers

### Tasks:
1. **Hero Section Background**
   - Add blue-tinted background section (similar to TeachCast)
   - Create visual container that separates hero from rest of page
   - Ensure text is readable on background (white text or high contrast)
   - Add subtle visual elements (gradients, patterns, or shapes)

2. **Competencies Section Background**
   - Use white/light background to contrast with hero
   - Add subtle background pattern or texture
   - Create clear visual boundary between hero and competencies

3. **Newsletter Section Background**
   - Use distinct background (light blue tint or white with accent)
   - Create visual separation from competencies section
   - Add subtle visual interest without distraction

4. **Section Dividers**
   - Add visual separators between major sections
   - Use subtle lines, gradients, or spacing
   - Ensure smooth transitions between sections

**Success Criteria:**
- Each section has distinct visual identity
- Clear visual hierarchy from top to bottom
- Sections don't blend together
- Professional, polished appearance

---

## Milestone 2: Content Organization & Visual Containers
**Goal:** Organize content into clear, visually distinct containers

### Tasks:
1. **Hero Content Container**
   - Wrap hero content in glassmorphic container
   - Add proper padding and spacing
   - Create visual focus point
   - Ensure content is centered and well-proportioned

2. **Competencies Cards Organization**
   - Group cards in clear visual container
   - Add background area for card section
   - Create visual grouping (not just floating cards)
   - Add section header with proper visual weight

3. **Newsletter Container**
   - Wrap newsletter in distinct visual container
   - Create clear call-to-action area
   - Add visual emphasis on form section
   - Ensure proper spacing and organization

4. **Content Max-Widths**
   - Standardize content widths across sections
   - Ensure proper reading width (not too wide)
   - Create visual rhythm with consistent widths

**Success Criteria:**
- Content is clearly organized in containers
- Visual hierarchy guides the eye
- No "text barfed up" feeling
- Professional, structured appearance

## Milestone 3: Typography & Content Hierarchy
**Goal:** Improve typography and create clear content hierarchy

### Tasks:
1. **Headline Hierarchy**
   - Ensure H1 is clearly the most prominent
   - H2 sections are clearly secondary
   - H3 card titles are properly weighted
   - Consistent typography scale throughout

2. **Text Organization**
   - Break up long paragraphs
   - Use proper line spacing and paragraph spacing
   - Create visual breathing room
   - Ensure text is scannable

3. **Content Grouping**
   - Group related content visually
   - Use spacing to create content blocks
   - Ensure logical flow of information
   - Create visual pauses between ideas

4. **Call-to-Action Emphasis**
   - Make buttons stand out visually
   - Create clear visual hierarchy for CTAs
   - Ensure buttons are properly sized and spaced
   - Add visual emphasis without being overwhelming

**Success Criteria:**
- Clear typography hierarchy
- Content is easy to scan and read
- Visual flow guides reading
- Professional typography throughout
