# Peliguard Website

A modernized website for Peliguard featuring glassmorphic design, built with Next.js 16.0.3 and optimized for Vercel deployment.

## Features

- **Glassmorphic Design**: Modern glassmorphism effects with backdrop blur and transparency
- **Hovering Navbar**: Navbar that gently slides down on hover or scroll
- **Professional Color Scheme**: Clean whites and blues (no gradients)
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Fast Performance**: Optimized for Next.js 16.0.3 with static generation

## Tech Stack

- Next.js 16.0.3
- React 19.2.0
- TypeScript
- Tailwind CSS 4
- Vercel-ready deployment

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm start
```

## Deployment

This project is ready for Vercel deployment. Simply connect your GitHub repository to Vercel, and it will automatically deploy.
